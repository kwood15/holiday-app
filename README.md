> Built with React and Express

## Get Started

`git clone` the repository

`cd` into the cloned directory

**Install the dependencies**:

```
npm install
```

**Run**

```
npm run dev
```

**Lint**

```
npm run lint
```

**Unit tests**

```
npm test
```

**E2E tests**

```
npm run test-e2e
```

**Build**

```
npm run build
```
